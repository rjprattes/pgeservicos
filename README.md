# Portal de Serviços da PGE

Plugin para GLPI desenvolvido para centralizar, organizar e facilitar o acesso aos serviços internos da Procuradoria-Geral do Estado do Espírito Santo - PGE-ES.

O plugin funciona como um portal institucional de serviços, reunindo em uma única interface áreas como Informática, Serviços Administrativos, Comissões, Reservas de Salas e Assessoria de Comunicação.

## Objetivo

Centralizar os serviços disponíveis aos usuários do GLPI, oferecendo uma tela inicial mais amigável, organizada por áreas de atendimento, com cards de acesso aos serviços, formulários, calendários e funcionalidades específicas.

O portal busca melhorar a experiência do usuário, reduzir dúvidas sobre onde abrir solicitações e facilitar o encaminhamento correto das demandas.

## Principais funcionalidades

- Portal inicial com cards por área de serviço;
- Área de Serviços de Informática integrada ao Form Creator;
- Área de Serviços Administrativos integrada ao Form Creator;
- Área de Serviços de Comissões;
- Área de Reserva de Salas de Reunião;
- Área de Serviços de Acessoria de Comunicação;
- Integração com formulários do GLPI/Form Creator;
- Redirecionamento automático para entidade correta ao abrir formulários;
- Redirecionamento automático para entidade correta ao abrir chamados vinculados;
- Customização visual dos formulários do Form Creator;
- Máscaras em campos de formulário;
- Calendário de reservas;
- Criação, edição e exclusão de reservas;
- Abertura e atualização de chamados vinculados às reservas;
- Organização modular de CSS e JavaScript.
